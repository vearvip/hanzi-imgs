# hanzi-imgs
所有汉字的图片（截止至Unicode15.1），采用jsdelivr作为cdn源
## 自行构建
安装依赖 
```bash
npm i
```
构建
```bash
npm run start
```
## cdn图片用法

### 采用天珩全字库（TH-Tshyn）

> 推荐

```js
https://cdn.jsdelivr.net/gh/vearvip/hanzi-imgs@v2/src/images/唯.png
```
